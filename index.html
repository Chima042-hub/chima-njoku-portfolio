Alert Triage

Suspicious PowerShell Activity on Windows Endpoint

Incident observed: Endpoint alert flagged encoded PowerShell execution from a user workstation during business hours.

Indicators: Base64 command pattern, unusual parent process chain, outbound connection attempt to an unknown IP.

Investigation steps: Reviewed process tree and command-line arguments, validated destination IP reputation, checked user login timeline and related host events.

Decision: Escalated as medium-severity suspicious execution, isolated host from network, and recommended credential reset plus malware scan.

Lessons learned: Parent-child process context and user activity timeline are critical for distinguishing admin scripts from malicious behavior.

Email Threat Analysis
Apr 2026

Suspicious Email Campaign with Lookalike Domain (Lab Scenario)

Scenario: In a lab dataset, multiple users reported an account-notification email that looked legitimate but arrived from a non-corporate sender.

Indicators: Domain typo-squatting, urgent language, shortened link, and mismatch between display name and sender address.

Investigation steps: Parsed email headers, expanded and sandbox-checked URL, compared domain age and WHOIS data, searched mailbox logs for similar messages.

Decision: Classified as malicious email activity in the simulation, blocked sender/domain, reported IOCs to detection lists, and issued user advisory with reporting guidance.

Lessons learned: Fast user reporting plus clear IOC documentation improves containment speed and reduces click-through risk.
