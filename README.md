## Local Run (Frontend)
Open index.html in your browser.

## Local Run (Security Tools)
1. Install dependencies:
   - pip install -r requirements.txt
2. Run security toolkit:
   - python security_toolkit.py
3. Run security training scripts:
   - python interactive_security.py

## Optional Local Backend (Flask)
- The backend in `portfolio_backend.py` is for local/demo API workflows.
- Start it with:
  - python portfolio_backend.py
- Security defaults:
  - Debug mode is OFF unless `FLASK_DEBUG=true` is set.
  - Admin messages endpoint requires `PORTFOLIO_ADMIN_TOKEN` and `X-Admin-Token` header.
